# Odin Links and Images Practice

This project is a practical exercise completed as part of [The Odin Project](https://www.theodinproject.com/) curriculum. It demonstrates the fundamental concepts of structuring web pages using HTML links and images.

## Features Covered

* **External Hyperlinks**: Implemented with security features like `target="_blank"` and `rel="noreferrer"`.
* **Internal Hyperlinks**: Configured relative paths to navigate between local project directory pages.
* **Absolute Image Paths**: Linked directly to hosting URLs like the official logo asset.
* **Relative Image Paths**: Hosted local image assets within a dedicated images subdirectory folder.
* **Accessibility Practices**: Included descriptive `alt` text tags for screen reader compatibility.
